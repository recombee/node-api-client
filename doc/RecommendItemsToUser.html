Class: RecommendItemsToUser

RecommendItemsToUser(userId, count, optional)

Based on user's past interactions (purchases, ratings, etc.) with the items, recommends top-N items that are most likely to be of high value for a given user. It is also possible to use POST HTTP method (for example in case of very long ReQL filter) - query parameters then become body parameters. The returned items are sorted by relevancy (first item being the most relevant).
